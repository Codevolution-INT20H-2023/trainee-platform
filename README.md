## Description

Web-app that helps future workers gain team-working expirience
## Installation

Clone the repository and execute:
```bash
$ npm install
```
Install husky:
```bash
$ npm run prepare
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run build
$ npm run start
```
